Controle de Estoque em VBA

Foi um projeto que eu peguei como Freelancer com o objetivo de auxiliar o cliente com o controle de estoque dele e realizar a integração com o Tiny ERP via API Rest.

O cliente me forneceu uma planilha e o formato que ele gostaria de visualizar as informações, sendo esta composta pelas tabelas: Enviar Produtos, Receber Produtos e Atualizar Informações.

Desenvolvi então em VBA um código para enviar produtos para o Tiny ERP, ao preencher algumas informações essenciais no excel e clicar em um botão, o produto era cadastrado no sistema com as informações preenchidas.

Logo após, desenvolvi o retorno de produtos que já foram baixados no sistema, clicando novamente em um botão os produtos retornavam para a planilha e se houvesse alguma alteração como no estoque, por exemplo, atualiza-va na planilha para ele conseguir visualizar.

E por último, uma tabela responsável pela alteração de dados dos produtos, como balancear estoque, trocar nome, código, inativar produto dentre outras funcionalidades.

Contudo, para auxiliar melhor a visualização do cliente, também coloquei para que o botão de Receber Produtos fosse ativado de tempo em tempo e de forma que só verificasse as alterações realizadas nos itens do sistema, proporcionando ao cliente um controle em tempo real de seu estoque.

A única limitação que encontrei no desenvolvimento desse projeto foi a falta de recursos do Tiny, onde não tinha por exemplo a API de Ordem de Compras.
